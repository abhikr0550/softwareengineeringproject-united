this(string musicfilepath, int channel)

Loads music file as a Mix_Chunk object so that it can be played in a specified audio channel. We play in a channel using SDL_Mixer so we can play multiple tunes and control each one separately. Chunks are still able to be looped by setting the loop value to -1.

void pauseMusic()

Pauses the audio file on the Music object's set audio channel. Uses SDL2_mixer Mix_Pause() method. An audio channel paused by Mix_Pause() can be resumed from the paused spot using Mix_Resume().
